# Debugging Prompts

## Root Cause Analysis

```
Debug this issue:
1. What is the expected behavior?
2. What is the actual behavior?
3. Trace the execution flow
4. Identify the root cause
5. Propose a fix with minimal side effects

Show your reasoning step by step.
```

## Error Investigation

```
I'm getting this error: [paste error]

1. Explain what this error means
2. List common causes
3. Check the relevant code paths
4. Identify the specific cause in this codebase
5. Provide a targeted fix

Don't suggest unrelated changes.
```

## Performance Debugging

```
This feature is slow:
1. Profile the execution path
2. Identify the slowest operations
3. Check for N+1 queries, excessive rebuilds, or blocking calls
4. Suggest specific optimizations
5. Estimate the impact of each fix

Focus on the top 3 improvements by impact.
```
