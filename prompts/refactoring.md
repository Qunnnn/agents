# Refactoring Prompts

## Extract & Decompose

```
Refactor this code by extracting reusable components:
1. Identify repeated patterns
2. Extract shared logic into utility functions/classes
3. Create proper abstractions
4. Maintain existing behavior (no functional changes)
5. Ensure all references are updated

Show me the before/after for each extraction.
```

## Modernize

```
Modernize this code:
1. Replace deprecated APIs with current equivalents
2. Use latest language features where appropriate
3. Improve type safety
4. Update to current best practices
5. Maintain backward compatibility

List each change with rationale.
```

## Performance

```
Optimize this code for performance:
1. Identify bottlenecks
2. Reduce unnecessary allocations
3. Optimize hot paths
4. Consider caching opportunities
5. Measure impact of each change

Prioritize by expected impact (high/medium/low).
```
