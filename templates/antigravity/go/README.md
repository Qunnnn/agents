# Antigravity Skills Template — Go Project

> Copy the `.agents/` folder into your Go project root.

## Structure

```
.agents/
├── skills/
│   ├── backend_go.md
│   └── api_handler.md
├── rules/
│   ├── coding_rules.md
│   └── agent_behavior.md
└── README.md
```

## Setup

```bash
# From your Go project root
cp -r /path/to/agents/templates/antigravity/go/.agents/ ./.agents/
```

## Customization

Edit the skill files under `.agents/skills/` to match your project's specific Clean Architecture conventions.
Add project-specific context (package names, API patterns, etc.) to make the AI more effective.
