# Antigravity Skills Template — Flutter Project

> Copy the `.agents/` folder into your Flutter project root.

## Structure

```
.agents/
├── skills/
│   ├── bloc_pattern.md
│   ├── localization.md
│   └── project_structure.md
├── rules/
│   ├── coding_rules.md
│   └── agent_behavior.md
└── README.md
```

## Setup

```bash
# From your Flutter project root
cp -r /path/to/Flutter-agent/templates/antigravity/flutter/.agents/ ./.agents/
```

## Customization

Edit the skill files under `.agents/skills/` to match your project's specific conventions.
Add project-specific context (package names, API patterns, etc.) to make the AI more effective.
