# Antigravity Skills Template — Swift/iOS Project

> Copy the `.agents/` folder into your iOS project root.

## Structure

```
.agents/
├── skills/
│   ├── ios_swift.md
│   └── architecture.md
├── rules/
│   ├── coding_rules.md
│   └── agent_behavior.md
└── README.md
```

## Setup

```bash
# From your iOS project root
cp -r /path/to/agents/templates/antigravity/swift/.agents/ ./.agents/
```

## Customization

Edit the skill files under `.agents/skills/` to match your project's specific conventions (e.g. MVVM, Feature modules).
Add project-specific context (package names, API patterns, DI Container, etc.) to make the AI more effective.
